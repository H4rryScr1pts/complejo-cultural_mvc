ComplejoCultural_MVC

Esto es un sistema para la administración de eventos y la  gestión de un estacinamiento para el Complejo Cultural Universitario, ubicado en Puebla, México.
Este proyecto cuenta con una zona pública, un area de usuarios y un area de administración.
